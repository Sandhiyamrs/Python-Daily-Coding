# WiFi Password Extractor

## Description
Extracts saved WiFi passwords from Windows system (requires administrator privileges).

## Features
- Lists all saved WiFi profiles
- Extracts passwords for each profile
- Exports to text file
- Cross-platform support (Windows only)

## Requirements
```
subprocess (built-in)
platform (built-in)
```

## Usage
```python
python file1_wifi_password_extractor.py
```

## Note
- Requires administrator/root privileges
- Windows only (uses netsh commands)

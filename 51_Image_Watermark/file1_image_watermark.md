# Image Watermark

## Description
Add text or image watermarks to your images with customizable position, opacity, and size.

## Features
- Text watermark with custom font and size
- Image watermark support
- Adjustable position and opacity
- Batch processing
- Multiple output formats

## Requirements
```
Pillow (install: pip install Pillow)
```

## Usage
```python
python file1_image_watermark.py
```

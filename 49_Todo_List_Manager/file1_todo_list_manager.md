# Todo List Manager

## Description
A CLI-based todo list manager with file persistence to track your daily tasks.

## Features
- Add, view, complete, and delete tasks
- Task priority levels (High, Medium, Low)
- Due date tracking
- Persistent storage (JSON file)
- Filter by status or priority

## Requirements
```
json (built-in)
datetime (built-in)
```

## Usage
```python
python file1_todo_list_manager.py
```

## How It Works
1. Stores tasks in a JSON file
2. Supports CRUD operations
3. Marks tasks as complete/incomplete
4. Filters and sorts tasks

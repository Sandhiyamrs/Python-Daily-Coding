# Port Scanner

## Description
A network port scanner to check open ports on a target host.

## Features
- Scan single or multiple ports
- Scan common ports or port ranges
- Multi-threaded scanning for speed
- Service detection
- Export results to file

## Requirements
```
socket (built-in)
threading (built-in)
```

## Usage
```python
python file1_port_scanner.py
```

## Note
- Use responsibly and only on networks you own or have permission to scan

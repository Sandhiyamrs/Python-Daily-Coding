# Morse Code Translator

## Description
Convert text to Morse code and vice versa with audio playback support.

## Features
- Text to Morse code
- Morse code to text
- Audio playback of Morse code
- Support for letters, numbers, and punctuation
- File input/output

## Requirements
```
winsound (built-in for Windows)
time (built-in)
```

## Usage
```python
python file1_morse_code_translator.py
```

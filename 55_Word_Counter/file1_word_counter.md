# Word Counter

## Description
Analyze text files and documents for word count, character count, reading time, and frequency analysis.

## Features
- Word and character count
- Line count
- Reading time estimation
- Most common words
- Vocabulary richness analysis
- Multiple file support

## Requirements
```
collections (built-in)
re (built-in)
```

## Usage
```python
python file1_word_counter.py
```

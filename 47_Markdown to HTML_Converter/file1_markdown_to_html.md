# Markdown to HTML Converter

## Description
Converts Markdown files to HTML with support for basic markdown syntax.

## Features
- Headers (H1-H6)
- Bold and italic text
- Links and images
- Lists (ordered and unordered)
- Code blocks
- Blockquotes

## Requirements
```
markdown (install: pip install markdown)
```

## Usage
```python
python file1_markdown_to_html.py
```

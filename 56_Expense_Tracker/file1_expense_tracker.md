# Expense Tracker

## Description
Personal finance manager to track income, expenses, and generate reports.

## Features
- Add/view/delete transactions
- Categorize expenses
- Monthly/yearly reports
- Budget tracking
- Export to CSV
- Visual summaries

## Requirements
```
json (built-in)
datetime (built-in)
csv (built-in)
```

## Usage
```python
python file1_expense_tracker.py
```
